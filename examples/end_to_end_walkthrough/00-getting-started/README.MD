# Cluster and Environment Setup - Amazon SageMaker HyperPod CLI Examples

This folder contains the following files:

- [00-setup.md](00-setup.md) - Instructions on how to install the CLI and setting up prerequisites you should have in your AWS account for running the examples.
- [01-(optional)-cluster-creation.md](01-(optional)-cluster-creation.md) - Optional instructions on how to create a new HyperPod cluster using the CLI. Alternatively an existing cluster can be used for the examples, or one can be created using the console UI.
